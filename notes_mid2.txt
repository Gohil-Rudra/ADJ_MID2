JSP directives tells the web container how to translate the JSP page into corresponding servlet :

JSP has 3 type of directives :

1 ) page -> import,info,extends,isErrorPage,errorPage
=> <% @ page ____ = "true/something_else" %>

2) <% @ include file = "______" %>

3) taglib -> to tag other libraries
